docker-freecad-cli
~~~~~~~~~~~~~~~~~~

Documentation
===================
https://wiki.freecadweb.org/FreeCAD_Docker_CLI_mode

How to run image?
====================

1. First, pull the image::

    docker pull ghcr.io/gigahawk/freecad-cli:latest

2. Run docker image::

    docker run -it ghcr.io/gigahawk/freecad-cli:lastest bash
